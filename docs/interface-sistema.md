# Interface do sistema

O sistema possui interface textual e apresenta um menu principal com opções para administrar categorias, fornecedores e produtos.

```text
=== DUPLATRABALHO | CONTROLE DE ESTOQUE ===
1 Cadastrar categoria | 2 Listar categorias | 3 Alterar categoria | 4 Excluir categoria
5 Cadastrar fornecedor | 6 Listar fornecedores | 7 Alterar fornecedor | 8 Excluir fornecedor
9 Cadastrar produto | 10 Listar produtos | 11 Alterar produto | 12 Excluir produto
0 Sair
```

Cada operação informa ao usuário se foi concluída com sucesso ou se ocorreu um erro de banco de dados.
